Privacy Policy

Effective Date: 29/08/25

This extension respects your privacy. We are committed to being transparent about what data is collected, how it is used, and ensuring that no unnecessary or sensitive information is ever stored.

Information We Collect

This extension only collects the minimum data required to provide its functionality:

  • User activity (limited to extension use): Trade events, price checks, and related interactions inside the EA Sports FC Web App.
  • Website content (limited to EA Sports FC Web App): Player data, tradepile data, and market information, in order to display prices, calculate profit, and provide trading insights.

Information We Do Not Collect

This extension does not collect or store:

  • Personally identifiable information (name, email, address, etc.)
  • Authentication data (passwords, PINs, or credentials)
  • Financial or payment information
  • Health information
  • Personal communications (emails, texts, chat messages)
  • Location or GPS data
  • Web browsing history outside of the EA Sports FC Web App

Data Storage

  • All data is stored locally using Chrome’s storage API where possible.
  • No personal or sensitive data is transmitted to external servers.
  • Any external API requests (e.g., for market prices) are limited to retrieving static FUT data and do not contain user information.

Use of Data

The collected data is used only to provide the extension’s features:

  • Displaying live player prices
  • Calculating trading profit and tax
  • Enhancing the EA Sports FC Web App experience

Third Parties

This extension does not sell, rent, or share any data with third parties.
